Simple Yet Powerful

Just three commands transform your workflow

1 Start Your Session

Begin each work session with a simple command. MSP instantly shows your current state, active tasks, and last progress. No more guessing where you left off.

.\msp.ps1 start

# Creates session entity
# Queries current state
# Shows active tasks
# Displays blockers

2 Track As You Work

Update progress naturally without leaving your flow. Every decision, breakthrough, and blocker is captured in real-time across all your tools.

.\msp.ps1 update "Fixed auth bug" 68
.\msp.ps1 decision "Chose JWT over sessions"
.\msp.ps1 blocker "API rate limit hit"

# Updates all tools automatically

3 End With Confidence

Close your session knowing everything is captured. MSP generates summaries, updates task boards, and ensures perfect handoff to your future self.

.\msp.ps1 end

# Finalizes session data
# Updates project state
# Syncs all tools
# Ready for next time
