Simple Setup (Recommended)

Download the Script

Just one file. That's it.

# PowerShell (Windows/Mac/Linux)
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/bleshinsky/sessionprotocol/main/msp.ps1" -OutFile "msp.ps1"

# Or with curl
curl -O https://raw.githubusercontent.com/bleshinsky/sessionprotocol/main/msp.ps1

Start Your First Session

Run this when you start working:

.\msp.ps1 start

You'll see:

üöÄ Session started at 09:15 AM
üìä Project: my-awesome-project
üìç Last session: Yesterday (2.5 hours)
üéØ Continue where you left off!

Track Your Progress

Update as you work (this is the mandatory part!):

# When you complete something
.\msp.ps1 update "Fixed user authentication bug" 15

# When you make a decision
.\msp.ps1 decide "Using JWT tokens instead of sessions"

# When you hit a blocker
.\msp.ps1 blocker "API rate limits need investigation"

End Your Session

Before you stop working:

.\msp.ps1 end

MSP saves your context and shows a summary:

‚úÖ Session complete!

That's It!

Your context is now saved. Tomorrow morning, just run msp start and you're back where you left off.
