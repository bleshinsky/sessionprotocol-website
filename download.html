Get Started with MSP

Complete installation guide - everything you need is free for personal use

Estimated setup time: 30 minutes. All tools have free versions perfect for MSP.
1

Install PowerShell 7+ (if needed)

Free Cross-platform

MSP requires PowerShell 7.0 or higher. Check your version:

$PSVersionTable.PSVersion

If you need to install or update:

Windows

winget install Microsoft.PowerShell

macOS

brew install powershell/tap/powershell

Linux

Visit Microsoft's Linux installation guide

2

Install Neo4j Desktop

Free for personal use No credit card required

Neo4j is a graph database that stores your project's knowledge graph.

  1. Visit neo4j.com/download
  2. Click "Download Neo4j Desktop"
  3. Fill the form (use personal email for free version)
  4. You'll receive an activation key via email
  5. Install and activate with your key

First-time Neo4j Setup:

  1. Create a new project called "MSP"
  2. Add a local DBMS (database)
  3. Set password (remember this for MSP config)
  4. Start the database
Default connection: bolt://localhost:7687
3

Install Obsidian

Free for personal use No account required

Obsidian is a markdown-based knowledge management tool.

  1. Visit obsidian.md/download
  2. Download for your platform
  3. Install and launch Obsidian
  4. Create a new vault called "MSP-Notes"
  5. Remember the vault location (you'll need this path)

Recommended Obsidian Settings:

  • Enable "Daily notes" plugin
  • Set daily note format: YYYY-MM-DD
  • Create folders: /daily, /decisions, /sessions
4

Create Linear Account

Free tier available No credit card required

Linear is a modern project management tool for tracking tasks.

  1. Visit linear.app
  2. Click "Get Started" (free)
  3. Sign up with email or Google
  4. Create a workspace (e.g., "Personal Projects")
  5. Create a team (e.g., "Development")

Get your API key:

  1. Go to Settings ‚Üí API ‚Üí Personal API keys
  2. Create new key named "MSP"
  3. Copy and save the key (shown only once!)
Save your Linear API key securely - you cannot view it again!
5

Install MSP

Open Source MIT License

Now that all prerequisites are installed, let's set up MSP:

# Clone the repository
git clone https://github.com/bleshinsky/sessionprotocol.git
cd sessionprotocol

# Run setup script
.\scripts\setup-msp.ps1

Configuration

The setup script will ask for:

  • Neo4j password (from step 2)
  • Obsidian vault path (from step 3)
  • Linear API key (from step 4)
6

Verify Installation

Test that everything is working:

# Run validation
.\msp.ps1 validate

# Start your first session
.\msp.ps1 start

Neo4j won't connect

  • Ensure database is running in Neo4j Desktop
  • Check bolt://localhost:7687 is accessible
  • Verify password in config

Obsidian vault not found

  • Use full absolute path to vault
  • On Windows: C:\Users\YourName\Documents\MSP-Notes
  • On Mac: /Users/YourName/Documents/MSP-Notes

Linear API errors

  • Regenerate API key if needed
  • Ensure key has full access permissions
  • Check internet connection
